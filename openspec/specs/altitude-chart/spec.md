# 海拔历史图表 (AltitudeChart)

## 概述

自定义 Canvas 绘制的迷你折线图，显示最近 60 秒的海拔变化趋势。

## 输入

| 参数 | 类型 | 说明 |
|------|------|------|
| history | List<Float> | 最近 60 个海拔值 (米) |
| modifier | Modifier | Compose 修饰符 |

## 数据源

- RingBuffer<Float>, 容量 60
- 每秒采样一次 (非每次 GPS 更新)
- ViewModel 中通过 `delay(1000L)` 循环采样

## 绘制规格

### 布局

```
┌──────────────────────────────────────────┐
│  ▁▂▃▅▆▇▅▆▇▆▅▃▂▁▂▃▅▆  ALT 60s         │
│  40m                               50m   │
└──────────────────────────────────────────┘
```

- 图表区域: 卡片内边距 12dp
- Y 轴标签: 左侧显示最小值, 右侧显示最大值
- 标题: 右上角 "ALT 60s", #6B6B80, 10sp

### 线条

- 贝塞尔曲线 (cubicTo), 平滑连接各点
- 线宽: 2dp
- 线色: #00E5FF (青色)
- 点: 不绘制单独的点, 只画线

### 填充

- 线条下方区域
- 渐变: 顶部 #00E5FF alpha 0.3 → 底部 #00E5FF alpha 0.0
- 使用 Brush.verticalGradient

### Y 轴自适应

- 最小值: history.min() - padding
- 最大值: history.max() + padding
- padding: (max - min) * 0.1, 最小 1m
- 当数据全相同时: 上下各 ±5m

### 网格线

- 水平虚线: 2-3 条 (根据 Y 轴范围自动计算)
- 颜色: #2A2A3A
- 线宽: 0.5dp
- 虚线间隔: 4dp

## 约束

- 最小高度: 80dp
- 最大高度: 120dp
- 数据不足 60 个点时: 左侧留空, 右侧对齐
- history 为空时: 显示空状态文字 "No data"
