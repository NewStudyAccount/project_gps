# 卫星信号面板 (SatellitePanel)

## 概述

可折叠的卫星信号展示面板，默认显示总览，可展开查看每颗卫星详情。

## 输入

| 参数 | 类型 | 说明 |
|------|------|------|
| satellites | List<SatelliteInfo> | 卫星列表 |
| expanded | Boolean | 是否展开 |
| onToggle | () -> Unit | 切换展开/折叠回调 |

## 折叠态

```
│  SAT 12/24  ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░  │
│  avg CN0: 38 dB  ·  tap to expand ▾ │
```

### 总览条

- 高度: 8dp
- 圆角: 4dp
- 背景: #2A2A3A
- 填充: 已用卫星数 / 可见卫星数 比例
- 填充色: 平均 CN0 对应颜色 (见下方阈值)
- 填充动画: 300ms 渐变

### 文字信息

- "SAT {inUse}/{total}": 14sp, #E0E0E0
- "avg CN0: {avg} dB": 12sp, #6B6B80
- "tap to expand ▾": 11sp, #6B6B80

## 展开态

```
│  SAT 12/24  ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░  │
│  ▴ collapse                         │
│  ┌────┬────┬────┬────┬────┬────┐   │
│  │ 01 │ 02 │ 03 │ 04 │ 05 │ 06 │   │
│  │ ██ │████│██▓ │ ██ │████│██▓ │   │
│  ├────┼────┼────┼────┼────┼────┤   │
│  │ 07 │ 08 │ 09 │ 10 │ 11 │ 12 │   │
│  │████│▓▓░ │████│██▓ │ ██ │████│   │
│  └────┴────┴────┴────┴────┴────┘   │
│  GPS: 8  GLO: 3  BDS: 1            │
```

### 卫星网格

- 列数: 6 列
- 每格: 卫星编号 + 信号强度柱
- 编号: 11sp, #E0E0E0, 居中
- 信号柱: 高度按 cn0 线性缩放
- 格子间距: 4dp
- 背景: #0A0A0F

### 信号强度柱

- 宽度: 固定 12dp
- 最大高度: 32dp
- 高度: `(cn0 / 50) * maxHeight`, 最小 2dp
- 圆角: 2dp
- 颜色阈值:
  - cn0 >= 35: #00FF6A (绿)
  - cn0 >= 20: #FFD700 (黄)
  - cn0 < 20: #FF3D00 (红)

### 底部统计

- 格式: "GPS: 8  GLO: 3  BDS: 1"
- 12sp, #6B6B80
- 按 Constellation 分组统计 inUse 的卫星

### 收起指示

- "tap to collapse ▴": 11sp, #6B6B80

## 动画

- 展开/折叠: `AnimatedVisibility`
  - enter: expandVertically + fadeIn, 250ms
  - exit: shrinkVertically + fadeOut, 250ms
- 总览条填充: 300ms 渐变

## 约束

- 折叠态高度: ~48dp
- 展开态最大高度: 屏幕剩余空间, 超出可滚动
- 卫星数为 0 时: 显示 "No satellites" 灰色文字
